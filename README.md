# IPL Data Analytics Project 🏏📊

## Overview
This project analyzes IPL match datasets using Python and data visualization libraries to uncover insights related to:
- Toss impact on match outcomes
- Team performance analysis
- Venue-wise chasing advantages
- Player of the Match statistics
- Yearly win percentage trends

## Technologies & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib


## Key Insights
- Certain IPL venues showed higher chasing success rates
- CSK and MI maintained strong long-term consistency
- Toss winning had moderate influence on match outcomes
- Some players dominated Player of the Match awards over multiple seasons

## Project Type
Exploratory Data Analysis (EDA)
